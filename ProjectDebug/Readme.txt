
************************* OVERVIEW ********************************
There are 2 Ways to run the tasks of ProjectBuild:

Without Debugging:
It's not necessary to run this app!
Simply double-click on task in the Task Runner Explorer

While Debugging:
Set this project (ProjectDebug) as the Startup Project
Select the ProjectDebug project in the Task Runner Explorer
Run this app by hitting F5
Set a break-point in the switch statement of the the taskList.ts file
Now, when you double-click on the task in the Task Runner Explorer,
you will hit your break point, and you will be able to step into the task

Gulp Tasks written in TypeScript
Full Intellisense
Setting BreakPoints and Debugging
